TalkDesign.org サイトへの導入

Q.

TalkDesign.org の目的は何ですか？

A.

TalkDesign.org は、知的設計（ID）運動の主張に対する回答を一元化して提供するために作成されました。ID に対する批判的な多くの記事は Web 上に散在しています。TalkDesign.org は、これらの記事の中で最も優れており最新のものへのリンク、およびこのサイトのために特に書かれた記事のコレクションを提供します。また、ID に関連する他の資料も提供しており、ID 擁護者のウェブサイトへのリンクも含まれています。

Q.

誰が TalkDesign.org を運営しているのですか？

A.

TalkDesign.org は 複数のボランティアによって運営されています 、さまざまな宗教的・哲学的な信念を持つ人々で、彼らはすべて ID の批判者であり、主流の進化生物学の支持者です。Talkdesign.org は、 TalkOrigins アーカイブ 、創造論と進化論の広範な論争に専念する長年のウェブサイトによって提供されています。テキサス州の 501(c)(3) 税務免除非営利団体である TalkOrigins アーカイブ財団は、TalkOrigins アーカイブ、TalkDesign、Panda's Thumb、および Antievolution.org のサポートを提供しています。

Q.

知的設計とは何ですか？

A.

ID 支持者の信念は大きく異なります。しかし、彼らがすべて共有していると思われる核心的な信念は以下の通りです：(a) 知的（おそらく意識のある）存在の行動が、生物の進化に関与した。 (b) この行動の存在を示す実証的証拠がすでに存在しており、そのような行動が発生したという科学的推論を正当化するのに十分なものである。用語「知的設計」は通常、これらの信念とそれらを支持する論理を指します。信念 (a) を持つが信念 (b) を持たない人々が、一般的に ID の支持者とはみなされないことは重要です。また、このウェブサイトはそのような人々と対立するものではありません。生物学における設計の実証的証拠が存在するという主張が論争を招き、私たちはこれを偽であるとみなしています。私たちは、この主張は疑似科学に基づいていると主張し、それが支持されているのは、その支持者の宗教的および/または思想的信念に訴えるからに過ぎないと考えます。

Q.

知的設計運動とは何ですか？

A.

知的設計（ID）運動は、進化論に対して宗教的（通常はキリスト教的）立場から反論する創造論の伝統から発展しました。ID の支持者はしばしば、彼らが単に「デザイナー」の存在を主張しているに過ぎないと主張しますが、すべての主要な支持者はデザイナーが神であると信じており、しばしば宗教的な問題の議論を伴って、科学的であると称する論拠を補強します。特に宗教的な聴衆に対処する際にはそうです。他の聴衆の前では、彼らのアジェンダの宗教的な側面を軽視します。弁護士であり創造論者のフィリップ・ジョンソンは、通常、「物質主義的科学」を転覆し、「神学的科学」に置き換えるという公言した意図を持って、知的設計運動を創設したと信賞されています。このアジェンダは現在、資金調達が良好な団体、すなわちディスカバリー・インスティテュートに属する科学と文化センター（CSC）によって積極的に追求されています。CSC は、 保守的なキリスト教徒 によって資金提供されています。（最近まで、CSC は科学と文化の再生センターとして知られていました；名称の変更は、おそらく彼らの思想的アジェンダをより明白にしないよう試みるものである。）現在、CSC は ID の促進において主導的な役割を果たしており、そのメンバーには ID の主要な支持者のほとんどが含まれています：ウィリアム・デムスキー、マイケル・ベヘ、ジョナサン・ウェルズ、スティーブン・マイヤーなど。彼らの 楔戦略 の目標は、ID が「科学における支配的な視点」になり、「私たちの宗教的、文化的、道徳的、政治的生活に浸透する」ことです。彼らの「科学的」論拠が主流の科学コミュニティ内で受け入れられる可能性が低いことを認識して、ID の支持者は、彼らの論拠を主に一般大衆、政治家、哲学者、そして他の非科学者に向けています。彼らが生産すると称される科学的な資料は、誤解を招く修辞、曖昧な用語、そして事実の歪曲で満ちています。彼らはまた、科学であることさえ目指していない多くの資料を生産しており、正直に言えばプロパガンダとして最もよく記述できるものです。

Q.

科学者は知的設計を支持しているのか？

A.

知的設計（ID）の擁護者は、自らが科学者の支持を得ているという印象を与えることに非常に熱心です。確かに、ID を支持する科学者も数多くおり、実際には若き地球創造論や他の多くの疑似科学を支持する科学者も存在します。しかし、彼らは科学者の総数に対して極めて少数であり、圧倒的な多数が進化論を支持しています。2001 年、ディスカバリー・インスティテュートは、100 名の科学者が署名したと発表するため、全国紙に広告を掲載しました。 声明 「ランダムな突然変異と自然選択が生命の複雑性を説明する能力に対する主張には懐疑的である」と述べている。署名者は ID を支持すると述べてはいないが、中には確かに支持している者もいる。これを以下と比較せよ。 手紙 進化の学校教育における現状の指導を支持し、80 の科学団体の会長によって署名されたものが議会に提出された。状況を比較対象として、進化の指導を公立学校で支持する団体である国立科学教育センター（NCSE）は、 プロジェクト・スティーブ , 上記で言及されたディスカバリー・インスティテュートによるものなどの反進化論リストへのパロディです。リストへの署名者は進化を支持し ID を拒否する声明に同意しましたが、一つ条件があります：すべての署名者は「スティーブ」またはそのバリエーションの名前です。スティーブが人口の約 1% を占めていることを考慮すると、300 人以上の署名者（執筆時点）は、ディスカバリー・インスティテュートに同意する科学者の 1 人に対して、おそらく数百人が反対していることを示唆しています。

Q.

自然主義に基づく ID への反対は正当化できますか？

A.

ID は、特に関連分野の科学者の大多数によって拒否されています。ID を受容する科学者の割合は無視できるほど小さいです。この分野の専門家による ID 論理の圧倒的な拒否を説明するために、ID 擁護者は人身攻撃の論法を用います。彼らは、科学者の大多数が「物質主義」または「自然主義」への固執によって偏っており、論理を公平に判断できないと非難します。しかし、これらの科学者の多くはそれ自体を神学者であるという事実にもかかわらずです。多くの（ただしすべてではない）主流の科学者と科学哲学者が、科学は「方法的自然主義」という原則に固執しなければならないと主張するのは事実です。これは、科学においてのみ「自然」な説明が認められると述べています。不幸にも、用語「自然」の意味は不明確です。これは通常、神の働きを伴ういかなる説明も排除すると想定されますが、経験的に検証可能であれば、神の働きを伴う仮説が「自然」として考慮される可能性があります。これらは曖昧な哲学的な水域であり、多くの科学者は単に、異常な主張には異常な証拠が必要であるというより一般的な原則に基づいて、方法的自然主義の原則を経験則として採用しているようです。より重要なのは、優れた科学的理論には検証可能性、簡潔性、説明力などの美徳が含まれることです。ID はこれらの美徳の多くを欠いており、他の多くの美徳も欠いています。「自然主義」の主張は、ID が基本的な科学的基準を満たすことが難しいという事実を隠すための口実となることがよくあります。いずれにせよ、ID 擁護者は、彼らの論理が神の設計者を意味しないことを保証します。設計者は宇宙人である可能性があります。方法的自然主義は確かにそのような設計者を排除しません。この問題に関する混乱は、言葉「自然」の曖昧さによって引き起こされました。「自然」は「人工的ではない」または「超自然的ではない」（他の可能な意味に加えて）を意味する可能性があります。ID 擁護者は、修辞的な目的のためにこれらの定義を頻繁に混同します。この問題は、マーク・イサクのエッセイ「『自然主義』の哲学的前提？」でより詳細に探求されています。 A Philosophical Premise of 'Naturalism'?

Q.

ID は創造論の一種ですか？

A.

この質問への答えは、一応「創造論」という言葉が何を意味するかによって部分的に異なります。スペクトルの一端では、創造論とは単に宇宙が神によって創造されたという信念であり、これはおそらくすべての一神教徒が共有する信念です。他端では、創世記の創造記述が文字通り真であり、科学的証拠がこの信念を支持するという若き地球創造論（Young Earth Creationism）です。合理的な中間的な定義は、個々の種や「種類」の動物が神によって創造されたという信念です。コアとなる ID（知的設計）の信念は、厳密には種の起源における神の関与を必然とはしませんが、すべての主要な ID 擁護者はそのような関信を信じています。多くの（おそらく大多数の）擁護者は、共通祖先説（common descent）、すなわち最も初期の生物から現在に至るまで親から子へと連続して継承される説を否定しています。CSC（Center for Science and Culture）の多くの資料は共通祖先説を攻撃しています。共通祖先説を否定する者には、フィリップ・ジョンソンとジョナサン・ウェルズが含まれます。デムスキーはこの問題に対して懐疑的であり、共通祖先説を否定することなく疑いを投げかけようとしています。CSC の少なくとも一人の会員であるポール・ネルソンは若き地球創造論者です。ID の戦略の一部は、最も極端な若き地球創造論者から進化論のほぼすべてを受け入れつつ進化が完全に自然であったという命題のみを拒否するマイケル・ビーチのような者まで、進化論の理論に対するすべての反対者が力を合わせられる「大きなテント」を作成することです。この大きなテントの統一性を維持するため、スペクトルの後端に近い者たちは、若き地球創造論者の最も著しい論理さえ批判することを避けることに注意を払っています。さらに、多くの ID の論理と戦術は、若き地球創造論者や古い地球創造論者のそれらと非常に似ています。情報理論や熱力学への無関係な訴え；純粋なランダムなタンパク質の組み合わせに基づく偽の確率計算；化石記録の欠如；文脈から切り取られた引用の使用；これらすべては創造論のメニューの定番です。ID の主要な特徴的な要素の一つは、議論の焦点を地球の歴史の詳細から「設計」や「目的論」といったより抽象的な概念へとシフトさせようとする試みです。これらは、非問答前提的な方法でほとんど使用されない用語です。若き地球創造論は非常に容易に偽証されるのに対し、ID は標準的な科学的実践に従ってテストするのが困難か不可能です。これにより、ID 運動は、テスト可能な肯定的な信念を持たない立場から議論できるという戦術的優位性を得て、同時に、進化論の理論のほぼあらゆる側面を攻撃できるようになります。ただし、これは ID を創造論よりも優れた理論にすることを意味するものではありません。多くの者は、これが ID をさらに非科学的にすると主張しています。また、既存の創造論運動とは（ある程度）異なる ID 運動を作成する動機の一部は、アメリカ合衆国における公立学校での創造論の教授に対する法的制限を回避するためであることも明確です。アメリカ合衆国最高裁判所は、「科学的」な創造論は宗教的な立場であり、したがって第一修正条項の教会と国家の憲法的分離に違反すると判決を下しています。創造論運動と ID 運動の間のより説得力のある共通点の一つは、科学コミュニティからの支持にもかかわらず、公立学校で彼らの見解を教えようとするという主要な焦点にあります。これら多くの ID と創造論の間のつながりが、ID を創造論の一種とみなすことを正当化するかは、最終的には個人の判断の問題です。多くの者は、そうであると結論付けており、ID は IDC、すなわち Intelligent Design Creationism（知的設計創造論）と呼ばれることがあります。ID の宗教的および政治的野心について時に率直である一方で、擁護者たちは他の時には ID が純粋な科学的問題であるという印象を与えるように試みます。Intelligent Design Creationism という用語の使用は、運動の真の性質に注意を向けさせるのに役立ちます。

Q.

知的設計は疑似科学ですか？

A.

私たちは、知的設計が若き地球創造論、占星術、アトランティス、そしてマスコミに溢れる科学を装う多くの他の信念システムと同様に疑似科学であると主張します。疑似科学とは、科学に基づいていると主張される信念システムですが、実際には論理的に欠陥があり、かつ非常に著しく悪質で、真摯な開かれた検討に耐えられないような論理に基づいているものを指します。疑似科学の擁護者は、自らの立場への教条的なコミットメントによって動機付けられており、これは彼らの心を反対する論理に対して閉ざします。疑似科学の一般的な（ただし普遍的ではない）追加症状には、以下のものがあります：論理は非科学者の聴衆に向けられる；大げさな主張がなされ、しばしば主張者が専門知識をほとんど持っていない分野で行われる（例：デムスキーの情報と熱力学の新しい法則への主張）；圧倒的な数の科学者たちが論理を公平に判断するために偏りすぎていると主張される；論理は余分な技術用語で表現される；論理は poorly defined な新しい技術用語（または古い用語に新しい意味を与える）を使用する；技術的論理は修辞に圧倒される；引用の過剰な使用があり、しばしば人気のある本からの引用であり、しばしば文脈から切り取られています。

Q.

知的設計を支持するために使用される「科学的」論理とは何ですか？

A.

知的設計（Intelligent Design）に対する論証は、主に「無知からの論証」、すなわち「神の隙間（god-of-the-gaps）論証」である。ID 擁護派はまた、「複雑で指定された情報（specified complexity）」や「不可約的な複雑性（irreducible complexity）」という形で ID の肯定的な証拠があると主張するが、これらの論証は実際には偽装された無知からの論証である。さらに、ID 擁護派は時に類推からの論証を行うこともある。彼らの多くの努力は、自らの ID 仮説を支持するのではなく、進化論（evolutionary theory）の特定の側面を攻撃することに費やされている。

Q.

無知からの論証、または神の隙間（god-of-the-gaps）論証とは何か？

A.

ID 擁護派は、生物学者によって特定の生物学的構造の進化が十分に説明されていないことを指摘する。これは真であり、当面の将来も真であり続けるだろう。なぜなら、我々はそのような構造に関する知識が現在非常に限られているからである。これに基づき、彼らはそれらの生物学的構造には進化論的な説明が存在しない、したがって知能あるエージェントによって設計されたものであると結論付ける。この論理の線は、時には彼らの仕事の中で明確に見られることもあるが、より頻繁には「不可約的な複雑性（irreducible complexity）」、「複雑で指定された情報（specified complexity）」や「情報」といった余分で誤解を招く用語を用いて論証を偽装している。

Q.

類推からの論証とは何か？

A.

類推による論証は通常、以下の通り展開される。生物学的システムは人工的な機械と共通の性質を有している。例えば、複数の協調する部分から構成されている。そのような機械の起源を直接観察した際、常に知的存在がその設計に責任を負っていた。したがって、知的存在が生物学的システムを設計したはずである。このような論証の一例は以下の通りである：「類推に基づいて結論を導くためには、帰納が共有された性質から流れ出るだけで十分である：不可約的な複雑さを持つルベ・ゴールドベルグ機械の製造には知的設計者の関与が必要であった；したがって、不可約的な複雑さを持つ血液凝固システムにも設計者の関与が必要であった。」（マイケル・ビー、『ダーウィンの黒い箱』、p. 218）類推による論証は、帰納論証の一種であり、著しく信頼性が低い。論証の重要性を説く側には、共有された性質の重要性と、相違する性質（あるいは類推の欠如）の無重要性を説得力を持って示す責任がある。実際、生物学的システムは人工的な機械とあらゆる点で大きく異なる。最も根本的な違いは、生物学的有機体が人工的な機械とは異なり、無数の世代にわたりランダムな変異と自然選択によって自己複製を続けてきたという点にある。このプロセスは適応をもたらすことが知られており、ある程度の自然進化による適応は、ID 擁護者ですら受け入れている。このような根本的な類推の欠如の面前で類推による論証を展開することは不合理である。ID 擁護者が行うように重要な類推の欠如を無視すれば、ばかげた結論に達するのは容易い。例えば、私たちが起源を直接観察した機械の設計に人間が責任を負っていた以上、生物学的「機械」も人間によって設計されたと推論すべきではないか。モンゴルフィエ兄弟の最初の気球飛行以前には、すべての翼のない生物がその時点で飛行できなかったため、人間も飛行できないはずであると推論していたかもしれない。（過去のある時点で、科学者たちは人間が飛行できないと考える正当な理由を持っていたかもしれないが、その理由は現在の物理学知識に基づいていたに過ぎず、ばかげた類推による論証に基づいていたわけではない。）

Q.

「不可約的な複雑性」とは何か？

A.

「不可約的な複雑性」という用語は、生化学者マイケル・ベヒーによって導入された。ID 擁護派は、不可約的に複雑な生化学系が自然進化によって（ある程度の合理的確率をもって）進化し得ないと主張する。不可約的な複雑性は当初、そのようなように定義され、系がどのように進化し得たかという考慮なしに、単に現在の系の状態を観察することによって検出できるとされた（前述の指定された複雑性の第 2 の意味に似ている）：もし生化学系の任意の部分を除去するとその系が機能しなくなるならば、それは不可約的に複雑であるとみなされた。この定義の認められた問題点は、ベヒーとデムスキーに新たな定義を提案させることになり、それはますます観察者が系が不可約的に複雑かどうかを判断する際に進化的前駆の可能性を考慮することを要求するようになった。これにより、系が不可約的に複雑かどうかを判断することがますます困難になった。しかし、利用可能な定義のいずれによって系が不可約的に複雑と判断されたとしても、進化の起源の可能性を排除するものではない。ベヒーとデムスキーの論理は、系が進化する過程で同じ機能を維持するという仮定に基づいている。しかし、生物学的系は進化するにつれてしばしば新しい機能に適応する。ベヒーは可能な進化経路を 2 つのカテゴリに分けた：機能の変化を伴わない「直接的」な経路と、伴う「間接的」な経路である。彼は次に、不可約的に複雑な系への「直接的」な進化経路の存立可能性に対する論理を展開した。また、間接的な経路による進化の確率は、これを許容可能な説明として受け入れるには低すぎる、と主張した。しかし、この主張は彼の直感と無知からの論理に過ぎないものであり、生物学者はそのような経路の詳細な説明をまだ提供していない。実際、生物学者はすでにベヒーが例として紹介したいくつかの系、例えば細菌鞭毛などの進化に関する説明を提案している。 免疫系 しかし、これらはおそらく、ベヘやデムスキーが要求する経路の各段階の正確な説明には十分ではないでしょう。彼らは、経路の各段階の正確な説明を要求するからです。それでも、これらの説明が徐々に詳細化していくにつれて、ID 擁護者はますますベヘの例の一つ、すなわち細菌鞭毛に集中する傾向にあります。これは、現時点では最も十分に説明されていない例である可能性があります。ベヘの論理に対する詳細な批判は、 ここ . 参照 不可約的な複雑性の解明 .

Q.

「指定された複雑性」とは何ですか？

A.

知的設計の擁護者は、「指定された複雑性」が知的設計の信頼できる指標であると主張しています。しかし、彼らはこの用語を全く異なる 2 つの意味で使用しており、これらを混同しがちです。「指定された複雑性」という用語は、生物学者レスリー・オルゲルによって考案されました。これは、生命体と非生命体を区別するものを特徴づけるためのものです。この用語は後に物理学者ポール・デイヴィスによって類似の形で採用・使用されました。これらの著者にとって、対象は圧縮してより短い記述にできない場合に「複雑」とみなされます。例えば、100 個の同一の数字「1111111111...」の列は、「100 個の 1」という記述に圧縮できるため、全く複雑ではありません。一方、100 個のランダムなバイナリ数字の列は、このように圧縮できないため複雑です。これは情報理論で広く使用されている概念、「アルゴリズム情報」または「コロモゴロフ複雑性」に対応します。ランダムな列はこの意味で非常に複雑であるため、複雑さだけでは生命を特徴づけることはできません。したがって、オルゲルとデイヴィスは「指定」または「特定性」の基準を追加しました。特定性は広く考えられることができます ここ 特定の性質が存在することを示唆している。デイヴィスは、生きている生物をコードする特別な配列セットのメンバーであるため「指定された」とされる DNA 配列の例を挙げている。オルゲルは以下のように要約している：簡潔に言えば、生きている生物は指定された複雑性によって区別される。結晶は、非常に多数の同一分子が均一に詰まっているため、単純でよく指定された構造の原型として一般的に受け入れられている。花崗岩の塊やランダムなポリマー混合物は、複雑ではあるが指定されていない構造の例である。結晶は複雑さを欠くため生物として認められず、ポリマー混合物は特異性を欠くため認められない。[Leslie Orgel, "The Origins of Life", Chapman & Hall, 1973.] 「指定された複雑性」という用語は、現在、数学者であり哲学者である知的設計（Intelligent Design）支持者のウィリアム・デムスキーによって再利用されている。しかし、彼は「複雑性」という言葉を全く異なる意味で使用している。デムスキーにとって、「複雑性」は単に起こる確率が低いことを再尺度化した測定値に過ぎない。ある物体が起こる確率が非常に低い場合、彼はそれが通常の意味で単純であっても複雑であっても、「複雑」とラベルを付ける。ある物体の起源が特定の仮説の下で十分に低い確率であることが示されれば、デムスキーは私たちにその仮説を物体の起源の説明として棄却するよう告げる。（確率を計算する際、私たちが観察した物体の特定の形態だけでなく、同じ「指定」に一致する他の形態も考慮しなければならない。例えば、同じ機能を果たす他の物体など；したがって「指定された」という言葉がある。）私たちが考えられるすべての自然（非設計）仮説を棄却でき、他の自然な説明がないと考える「説得力のある理由」がある場合、デムスキーは私たちにその物体が指定された複雑性を示していると告げ、それが設計された結果であると推論すべきだと述べる。例えば、デムスキーは民主党の政治家ニコラス・カプートが選挙の投票順を決定したケースを説明している。41 回の選挙のうち 40 回、カプートは民主党候補者に有利と知られていた投票順の上位に民主党候補者を置いた。デムスキーはこの出来事を、以下の 40 個の D と 1 個の R の配列で表している：DDDDDDDDDDDDDDDDDDDDDRDDDDDDDDDDDDDDDDDD カプートは、投票順を決定するために壺からカプセルを引いたと主張し、民主党と共和党に同様に上位に立つ機会を与えたという。この仮説が真である場合、デムスキーは 40 個以上の D を引く確率は約 500 億分の 1（「40 個以上の D」が指定）であると計算し、この確率が仮説を棄却するのに十分低いと判断する。さらに、彼は「壺モデルは利用可能な最も信頼性の高いランダム化技術のうちの 1 つである」という理由により、他の自然な説明がないと考える「説得力のある理由」があるとも主張している。したがって、彼は上記の配列が指定された複雑性を示し、設計の結果であると結論づける。つまり、デムスキーの意味では、この配列のような非常に単純な現象でも指定された複雑性を示すことができる。（明らかに、カプートが民主党を 41 回すべてで最初に置いた場合、デムスキーのアプローチに従って設計を推論する理由はさらに大きくなり、41 個の D の配列も指定された複雑性を示したであろう。）不幸なことに、デムスキーは「指定された複雑性」という用語の自分の使用法をオルゲルとデイヴィスのそれと混同しているが、両者はかなり異なる。多くの生物学者は、オルゲルとデイヴィスの意味で生きている生物が指定された複雑性を示していると同意するだろう。しかし、彼らはデムスキーの意味で生きている生物が指定された複雑性を示していると同意しないだろう。なぜなら、彼らは生物学的構造が自然に進化しなかったと考える「説得力のある理由」をデムスキーが与えたことを受け入れないからである。上記はデムスキーが指定された複雑性によって何を意味するかという解釈である。しかし、彼はこの問題について極めて曖昧で二重の意味を持ち、正確な定義に固定されることを拒否している。したがって、彼が何を意味するかを確実には誰も言うことはできない。

Q.

デムスキーは生物学における設計の推論に「指定された複雑性」の方法をどのように適用したのか？

A.

デムスキーは、彼の方法を生物学に適用する詳細なケースとして、大腸菌（E. coli）の鞭毛（flagellum）にのみ提供している。これは彼の最新の書籍『No Free Lunch』に現れる。この場合、デムスキーが考慮する自然な仮説は、鞭毛がタンパク質の純粋なランダムな組み合わせの結果として出現したという仮説である。これは、ランダムでない効果を考慮していない。最も重要なのは、進化論の中心的な原理である自然選択（natural selection）を無視していることである。純粋なランダムな組み合わせの仮説はすでに生物学者によって普遍的に拒絶されている（それは古い創造論者の「ジャンクヤードの竜巻」の稻草人である）ので、デムスキーがこの仮説を検討することは有用な機能を持たず、仮説を棄却するために使用される確率計算は無関係である。デムスキーの確率計算が無関係であるため、本当に重要なのは、他の自然な説明がないと考える彼の「説得力のある理由」（彼はこれを「禁止的一般化」と呼んでいる）である。これは、不可約的な複雑性（irreducible complexity）からの議論の形をとるため、すべての「指定された複雑性」に関する誤解を招く用語が解きほぐされたとき、デムスキーは単に別の不可約的な複雑性からの議論をしているだけであることがわかる。彼はビヘーの議論に何を足したのか？ほとんど何もない。彼はビヘーの「直接的」な進化経路に対する議論を強化したが、鞭毛が異なる機能を持つシステムから進化した可能性を再び考慮していない。機能の変化は進化において一般的であり、「共利用（co-option）」または「共適応（co-optation）」と呼ばれることが多い。デムスキーは共利用について言及しているが、個々のタンパク質が共利用される可能性のみを考慮している。生物学者が実際に提案しているシナリオ、すなわち多数のタンパク質からなるシステムの大部分を単位として共利用するシナリオを無視している。生物学者は、鞭毛が分泌システムから進化したと提案している。実際、現代の鞭毛は依然として分泌システムの機能を保持している。このシナリオがビヘーとデムスキーが要求する詳細のレベルまで発展していないことは事実だが、デムスキーが生物学者が鞭毛が進化した方法について「何も知らない」と主張することは、彼が行うように非常に誤解を招く。デムスキーの議論に対する詳細な批判は、 here .

Q.

「情報」がどのように図式に組み込まれるのか？

A.

知的設計の擁護者は、しばしば「情報」という曖昧な用語に訴え、生物に存在する情報は知的設計によってのみ生成されることを主張します。しかし、この用語が何を意味するかを明確に定義せずにこれらの主張を行うことは無意味です。情報理論の分野では、いくつかの異なる「情報」の定義が用いられており、その中で最も一般的な 2 つはアルゴリズム情報とシャノン情報です。前者はコルモゴロフ複雑性としても知られており、上記で簡潔に説明されています。後者は主に通信において用いられる確率的概念であり、メッセージの伝送効率と信頼性を最大化することを目的としています。メッセージの意味は重要ではありません。これらの情報の両方が自然過程によって生成されることが示されています。デムスキーは情報からの論証を形式化しようとしましたが、彼の定式化は結局のところ彼の複雑性からの論証と同じであることが判明しました。彼は「情報」と「複雑性」という用語を同義語として使用しています。どちらも単に不確実性の尺度を再尺度化したものに過ぎません。デムスキーが事象が高い情報量を示すと述べる場合、それはその事象が低い確率を持つことを意味します。彼の「複雑で指定された情報」は彼の「指定された複雑性」と全く同じであり、彼はこれら 2 つの用語を相互に使用しています。参照 創造論と情報理論 を参照してください。
