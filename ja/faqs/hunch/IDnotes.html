[1] 数千人の科学者が ID 運動に参加し、残りの人々もやがて目覚めるという主張は、 必勝というプロパガンダ技術であり、ID 運動によって頻繁に用いられています。そのため、ID 派のプレスリリースや新聞記事が年々、同じ少数の人名を繰り返し言及しているのは非常に奇妙です。特にデムスキー、ウェルズ、ビヘの名前が挙げられ、まるで彼らが数千人の参加者の中から偶然選ばれたサンプルであるかのように扱われています。（明確にするために言っておくと、この三人の中で科学者と呼ぶにふさわしいのはビヘだけです。）ID 派/創造論者の即ち成功を宣言する例を多数確認するには、 創造論における最も長期間にわたる虚偽：進化の即ち崩壊 グレン・モートンによる：


「最近、デムスキーや他の ID 支持者の著作を読みましたが、彼らが 40 年以上前から主張しているような主張を見つけました。これは若き地論者が長年主張してきたものです。その主張とは、進化論（またはその主要な支持概念）が科学者によってますます放棄されようとしている、あるいは崩壊の瀬戸際にあるというものです。この主張には多くの形態があり、162 年以上にわたって主張されてきました。」

[2] ID 派は、彼らが創造論者ではない、さらには反進化論者でもないことを主張することで、自らのイメージを守ろうと熱心です。しかし、その最新の公衆学校への ID 導入のための手口は、「進化に対する証拠を教える」ことです。オハイオ州で行われた世論調査について彼らが誇らしげに語ったことを考えてみてください： ダーウィンはこの議論を愛したでしょう:

「どの選択肢（「A」か「B」）があなたの見解を表していますか？」

A. 生物の教師は、ダーウィンの進化論とそれを支持する科学的証拠のみを教えるべきである。

B. 生物の教師は、ダーウィンの進化論を教え、それに対する科学的証拠も教えるべきである。

2001 年のゾグビー調査によると、全国成人の 15% だけが「A」に同意し、71% が「B」に同意している。（未確定：14%）

これは、実際にはそうではないように人々を思い込ませるため、典型的な誘導質問の完璧な例です。 です 進化に対する証拠についてであり、これが真実であれば、それを教えることがなぜ非難されるのか？ ID 主義者たちは、彼らが「起源科学」と呼んでいるものが「客観的」に教えられなければならないと主張する。彼らが「客観的」と言うとき、彼らが本当に意味しているのは、ID の論証に同様の重みを与えることである。まるで、真に客観的かつ知識豊富な人がそれらを正当なものとして見出すだろうと考えるように。ジョージ・オーウェルは誇りに思うだろう。しかし、ある意味では、彼らは一点において正しい：ID を「進化に対する証拠」と呼ぶのは、ある程度適切である。なぜなら、それは単なる批判の集まりであり、しかも悪い批判に過ぎないからだ。

[3] IDist の信念には、運動の著名な指導者間であっても、非常に一貫性が乏しい。例えば、マイケル・ベヒーは古代の地球と共通祖先説を認めるが、自然過程が進化のすべてを説明できるとは信じていない。一方、フィリップ・ジョンソンは共通祖先説を頑なに否定している。ポール・ネルソンはさらに対照的に、若き地球創造論者である。明らかに、これらの人々のうち少なくとも二人は、信念の主要な側面の少なくとも一つについて大いに間違っているにもかかわらず、ID 運動はこれらの違いを軽視するためにできる限りのことをしている。驚くべきことに、彼らは、これら科学的問題に関する結論を導き出すこと（これこそが彼らがずっと議論してきたことである）自体が、 関連性あり！

私の知る限り、ベヒーは明確に進化を肯定する主要な ID 主義者の唯一の人である（ただし、彼はその進化論を否定する人々に対して称賛を惜しまない）。他の多くの人々は安全に創造論者に分類できるか、あるいは彼らが何を信じているかを誰も正確に知るにはあまりにも慎重すぎる。しかし、彼らを創造論者と呼んではならない——彼らは狂乱し、あなたが誤った表現をしていると告発するだろう。実際、ID 運動は、「創造論と結びつけることで彼らを迫害し、信用を失わせようとする『独断的なダーウィン主義者』によってステレオタイプ化されている」と主張することで、多くの修辞的な効果を得ている。しかし、彼らの大多数が です 創造論者たちにとって、ID 運動が本質的に同じ戦術を用い、旧来の運動と全く同じ上層目標（宗教的弁明と「文化的再生」）を有していることも事実である。

楔（ウェッジ）戦略は純粋に政治的なものだ。 真の科学的運動は、そのメンバー間の違いを激しく議論するはずであり、進化生物学者もそうしている。 皮肉なことに、ID 主義者が何を、あるいは何を拒否しているのかを知ることは難しい。 彼らは具体的な詳細を語ろうとしないからだ。 これにより、議論において極めて滑りやすい立場に置かれるという追加のメリットがある。 これについては ジョンソンとのインタビュー を参照:

"だから問題はこうなる：『どうすれば勝てるのか？』 これが私が、今『楔（ウェッジ）』戦略として完全に展開されているものを開発し始めた時期だ。 『最も重要なことに集中せよ』−−つまり、メカニズムと情報の蓄積だ。 聖書と創世記を議論から除外せよ。 なぜなら、いわゆる聖書と科学の対立を招きたくないからだ。 議論を世俗の学界で受け入れられるように、かつ宗教的な異議申し立て者を統一する方向にまとめるように言い換えよ。 つまり、『創造主が必要なのか、それとも自然がそれだけでできるのか？』と集中し、人々がいつも試みに誘導しようとする他の問題には深入りしないことだ。 彼らは『ノアの洪水についてどう思うか？』などと尋ねてくるだろう。 このような質問には決して乗らない。 なぜなら、それは道のない荒野にあなたを連れて行き、そこから決して抜け出せないからだ。"
