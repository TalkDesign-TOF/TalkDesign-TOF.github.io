ID 主義者たち...

知的設計について...

  • ID は我々がそう定義するものであり、我々は合意していません。

  • 科学者の数も年々増加し、ID 運動に参加しています。そのため、毎年同じ 3 つの理由を挙げています。[1]

  • ID は創造論ではなく、進化と完全に両立可能です。これが、学校に「進化に対する証拠」を教えるよう求める理由です。[2]

  • 私たちは創造論者ではありません。ただし、私たちがそうである者もいれば、そうでない者もいます。しかし、残りの私たちは、自分が創造論者ではないことを確認することはありません。あなたがたが私たちを創造論者と呼べば、私たちは文句を言うでしょう。[3]

  • 地球の古さ、生物の共通祖先、自然選択といった問題に対する正しい立場は、まず一般大衆に、これら少なくとも一つを拒否すべきだと納得させた後で、検討することができます。[4]

  • ID（知的設計）は科学コミュニティで広く受け入れられている理論です。たった去年、100 人以上の科学者が ID を支持しない声明に署名しましたが、それらはダーウィニズムに対して「懐疑的」であると述べています。何万人もの他の科学者の意見は数えられません。なぜなら、彼らはすべて偏見を持っているからです。[5]

  • ID は、設計の科学に関する研究のためのプログラムに過ぎません。私たちの研究計画の一部には、幼児向けの塗り絵を作成したり、パーティでより好かれるようになりたりすることです。[6]

  • ID は、自然における目的と目的論を検出するための科学的理論です。しかし、その目的が何かを尋ねることはしないでください。なぜなら、それは ID とは別の宗教的な問題だからです。

  • 設計者は神から宇宙人まで何でもあり得ます。しかし、それが宇宙人だと信じるラエリアンたちは、非論理的です。

ダーウィニズムについて...

  • ダーウィニズムは生命の進化をあらゆる細部で説明できないため、それは間違っています。しかし、ID 主義者にこれらのことを説明させることはしないでください。なぜなら、ID はそのような種類の理論ではないからです。[7]

  • 主流の科学者たちは、ダーウィニズムの正統性を象徴する巨大なブロックに異議を唱えることを恐れています。しかし、ここでは相互に意見が異なる主流の科学者たちがいます。これは、彼らが何も合意できないことを意味します。[8]

  • ダーウィニストたちは宗教的・思想的動機によって駆動されています。しかし、私たちがウェブサイトから神の絵と「文化的再生」というフレーズを削除したため、誰もがこれが私たちには当てはまらないことを知っています。[9]

  • 社会におけるすべての誤りは、独断的なダーウィニズム的無神論的唯物論者によって引き起こされています。ステレオタイプ化、悪魔化、および替罪羊化も含まれます。[10]

  • ダーウィニストたちは社会主義と自由放任資本主義の両方、人種差別とリベラリズムの両方、フェミニズムとセクシズムの両方、動物実験と動物権利運動の両方、そして共産主義ナチズムにも責任があります。[11]

哲学について...

  • 哲学者たちは、科学と非科学の境界がどこにあるかを正確に合意できません。したがって、私たちがそう言うなら、何でも科学とみなすことができます。

  • もし生きているシステムが巧みに設計されているように見えるなら、それは ID の証拠です。もし拙劣に設計されているように見えるなら、それは単に私たちが良い設計と悪い設計を構成するものをどのように知る方法もないからです。

  • 結局のところ、設計者が本当に意図していることを知る方法がないため、それが悪い設計であるかどうかを判断することはできません。しかし、設計者が意図するものによると、ID は文化、社会、法律を革命化すると私たちは知っています。[14]

  • 方法的自然主義は、超自然的な説明を検討することを妨げる不公平なルールである。 しかし、これは探偵が人の死において知的な主体を考慮できないことを意味する。 なぜなら、私たちが皆知っているように、殺人犯は超自然的だからだ。 [15]

  • ID といった優れた科学的理論は、曖昧で不明確であり、メカニズムや歴史に関する具体的な詳細を提案することを拒否すべきである。 未特定の実体がある時点で、どこかで、何らかの方法で「設計」した、というものは、十分に妥当な説明である。

  • 設計からの論証は神学的な論証ではない。 なぜなら、私たちは必ずしも神について話しているわけではないからだ。 しかし、設計からの論証に対する反論は神学的である。 なぜなら、それは「神はそれをこのようにしないだろう」と私たちに言わなければならないことを要求するからだ。 これは正当ではない。 [16]

証拠について...

  • コナジラミのケースが問題があると証明されたため、自然選択は否定された。 野外における自然選択の他の 1,582 の研究、ならびに多数の实验室研究は、カウントされない。 [17]

  • そして、コナジラミは樹幹に休まない。 野外の自然な位置で見つかったコナジラミの実際のデータセット、つまり樹幹から離れているが樹幹上にあるものも含まれるが、研究者たちは何年もの間、コナジラミのトラップで数千匹のコナジラミを捕獲しており、トラップに入ったコナジラミが一度も樹幹上で見つかったことはないため、無関係である。18]

  • コナジラミは樹幹に休むのではなく、枝の上部に休むため、鳥はそれらに到達できない。 なぜなら、鳥が枝を訪れることを防ぐ魔法の障壁があるからだ。

  • 上記で示されたように、コナジラミは樹幹に休まないため、教科書に見つかる樹幹上のコナジラミの対照的な目立たしさを示す写真は、詐欺、詐欺、詐欺である。 しかし、教科書にある他の staged 動物写真は、異議なしである。

  • より包括的な分類群（例えば門）が、より特定の分類群（例えば属）よりも先に出現したという事実は、進化論に対する証拠である。同様に、ヨーロッパ人がトニー・ブレアよりも先に出現したという事実は、人類の共通祖先説に対する証拠である。[19]

  • 進化は新たな情報を生み出すことができない。なぜなら、酵素の基質特異性を高めるような変化は、酵素が他の化合物との反応性を低下させることを意味し、これは情報の損失だからである。同様に、酵素の汎用性を高めるような変化も、酵素が特定の性質を失ったため、情報の損失である。[20]

  • 生命は自然な手段によって生じることはできない。それは指定された複雑性（Specified Complexity）を有しているからである。指定された複雑性とは、自然な手段によって生じることはできないものを指すため、生命は指定された複雑性を示さなければならない。[21]

  • 私たちが愛する設計者によって、私たちが設計された致命的な病気から守るための免疫系が設計されたことは、とても素晴らしいことです。

  • 生きているものの根本的な統一性は、設計者はただ一人であることを意味する。生きているものの間の異常な多様性、それらがお互いを殺す傾向を含む、単一の設計者が非常に創造的かつ気まぐれであることを意味するに過ぎない。[22]

  • 横方向の遺伝子移動は、進化の強力なメカニズムであるが、進化に対する証拠である。

  • 宇宙の法則が生命に完璧に適していることは、設計者の証拠である。宇宙の法則が生命を生み出すことができないことも、設計者の証拠である。[23]

  • 不可約的な複雑性を有する構造は複数の部品を必要とする。したがって、それらは進化することはできない。もし誰かが、複数の部品を必要とする構造がどのように進化し得たかを示したとしても、それは単にその構造が「不可約的な複雑性」を有していなかったことを意味するに過ぎない。

  • IC 構造は分子系である必要があります。マウストラップを除く。

  • 「間接的な」経路は極めて稀であり、レプリーチャンのように見つけるのが難しく、したがって「最小限の」可能性にはなりますが現実的なものではありません。そのため、ダーウィンからダウキンスに至るまで、主要な生物学者によって詳細に検討されたにもかかわらず、安全に無視することができます。[26]

  • 一方、ID 仮説はレプリーチャンのようなものではありません。
