"The Quixotic Message" 或 "No Free Hunch"

Steve Reuland（又名 "theyeti"）以幽默但引用详实的方式审视了智能设计运动及其倡导者之间的矛盾之处。

by theyeti *

[贡献于 2003 年 1 月 22 日]

[作者注：本集矛盾与荒谬之处最初是在 antievolution.org 讨论板上撰写并发布的。后来决定将其发布到网上，以便让更广泛的受众看到。本集旨在轻松幽默，但也突显了 ID 辩论中的一些严肃问题。如果这让你感到不适，那么请去读些别的东西。此外，应几位读者的敦促，我决定制作一个参考文献页面来记录这些不一致之处。可以通过点击下方任意一个编号的参考文献来访问。如果你觉得这会破坏幽默感，那么请随意跳过。但正如 Dave Barry 所说， I'm not making this up...]

IDists...

On Intelligent Design...

  • ID 是我们说它是什么，而且我们并不一致。

  • 越来越多的科学家加入了 ID 运动，这就是为什么我们每年都在重复同样的三件事。[1]

  • ID 不是创造论，并且可以与进化论完美兼容。这就是为什么我们要要求学校教授“反对进化论的证据”。[2]

  • 我们并非创造论者，除了那些自认是创造论者的人；但其余的人也不会承认自己不是。然而，若有人称我们为创造论者，我们将不胜其烦。[3]

  • 关于地球古老性、生物共同祖先以及自然选择等问题，正确的立场可以在稍后确定，前提是我们已说服公众，他们应当至少拒绝其中一项。[4]

  • 智能设计（ID）是科学界广泛接受的理论。就在去年，超过 100 位科学家签署了一份声明，该声明不支持智能设计，但表示他们对达尔文主义持“怀疑”态度。其他成千上万科学家的观点并不被计入，因为他们都是有偏见的。[5]

  • 智能设计（ID）是一个关于设计科学的科研项目，仅此而已。我们研究计划的一部分包括为学龄前儿童制作涂色书，以及在聚会上让自己更讨人喜欢。[6]

  • 智能设计（ID）是一种用于在自然界中检测目的论的科学理论。但请不要询问我们那个目的是什么，因为那是一个与智能设计无关的宗教问题。

  • 设计者可能是从上帝到外星人的任何存在。但认为设计者是外星人的拉利安人，其推理是不合逻辑的。

关于达尔文主义……

  • 达尔文主义无法解释生命演化的每一个细节，因此它是错误的。但请不要要求智能设计论者解释这些事情，因为智能设计并非此类理论。[7]

  • 主流科学家不敢与达尔文正统这一铁板一块唱反调。然而，这里有一些主流科学家在某些问题上互不相让，这意味着他们无法达成一致。[8]

  • 达尔文主义者受宗教和意识形态动机驱使。但既然我们从网站上移除了上帝的图片和“文化复兴”的短语，大家就知道这不适用于我们。[9]

  • 社会中的一切错误都是由教条式的达尔文主义无神论唯物主义者造成的。包括刻板印象、妖魔化和替罪羊。[10]

  • 达尔文主义者既对社会主义和自由放任资本主义负责，也对种族主义和自由主义负责。既对女权主义和性别歧视负责，也对动物实验和动物权利运动负责。还有共产主义纳粹主义。[11]

在哲学方面...

  • 哲学家无法就科学与非科学之间的确切界限达成一致。因此，如果我们说某事是科学，那么任何事都可以被认为是科学。

  • 如果一个生命系统看起来设计良好，那就是智能设计（ID）的证据。如果它看起来设计拙劣，那只是因为我们无法知道什么是好设计什么是坏设计。

  • 毕竟，我们无法判断那是坏设计，因为我们无法知道设计者真正的意图。但我们确实知道，根据设计者的意图，智能设计将彻底改变文化、社会和法律。[14]

  • 方法论自然主义是一条不公平的规则，阻碍我们考虑超自然解释。但这意味着侦探在调查人员死亡时不能考虑智能代理，因为众所周知，凶手是超自然的。[15]

  • 像 ID 这样良好的科学理论应该是模糊和模棱两可的，并拒绝提出任何关于机制或历史的具体细节。某个未指明的存在在某个时间、某个地方、以某种方式‘设计’了某物，这是一个完全合理的解释。

  • 设计论证不是一个神学论证，因为我们不一定在谈论上帝。但任何对设计论证的反驳都是神学的，因为它要求我们说‘上帝不会这样做’，这是不合法的。[16]

关于证据...

  • 由于胡椒蛾案例已被证明存在问题，自然选择被驳斥了。野外对自然选择的另外 1,582 项研究，以及大量的实验室研究，都不算数。17]

  • 而且胡椒蛾并不栖息在树干上。在野外自然位置发现的蛾子的实际数据集，无论是在树干上还是不在树干上，都是无关紧要的，因为研究人员多年来一直在他们的蛾子陷阱中捕获了数千只蛾子，从未发现过一只在陷阱中的蛾子栖息在树干上。18]

  • 由于蛾子不栖息在树干上，而是栖息在树枝较高的地方，这意味着鸟类无法到达它们，因为有一个魔法屏障阻止鸟类访问树枝。

  • 如上所示，蛾子不栖息在树干上，这意味着教科书中显示树干上蛾子对比显著性的照片是欺诈，欺诈，欺诈。然而，教科书中所有其他摆拍动物照片都是无可指责的。

  • 更广泛的分类群（如门）出现在更具体的分类群（如属）之前，这构成了反对进化的证据。同样，欧洲人出现在托尼·布莱尔之前这一事实，也是反对人类共同祖先的证据。[19]

  • 进化无法产生新信息，因为任何提高酶底物特异性的改变都会降低该酶与其他化合物的反应性，这构成信息的损失。同样，任何提高酶通用性的改变也是信息的损失，因为酶失去了部分特异性。[20]

  • 生命不可能通过自然方式产生，因为它具有指定复杂性。指定复杂性意味着某事物无法通过自然方式产生，因此生命必须表现出指定复杂性。[21]

  • 我们慈爱的设计者为我们设计了一个免疫系统，以保护我们免受他所设计的致命疾病的侵害。

  • 生物的基本统一性意味着只有一位设计者。生物之间非凡的变异，包括它们相互杀戮的倾向，仅仅意味着我们这位独一无二的设计者极具创造力且变幻莫测。[22]

  • 横向基因转移，作为一种强大的进化机制，却是反对进化的证据。

  • 宇宙定律完美适合生命，这是存在设计者的证据。宇宙定律无法产生生命，这也是存在设计者的证据。[23]

  • 不可还原复杂的结构需要多个部分。因此，它们无法进化。如果有人证明了一个需要多个部分的结构是如何进化的，那仅仅意味着该结构并非不可还原复杂。

  • IC 结构必须是分子系统。除了捕鼠器。

  • "间接"路径极其不可能，且难以发现，就像找到小精灵一样，因此它们仅仅是"勉强"的可能性，而非现实的可能性，尽管从达尔文到道金斯的所有主要生物学家都对其给予了详细关注，但仍可安全地忽略。[26]

  • 另一方面，ID 假设与小精灵毫无相似之处。
