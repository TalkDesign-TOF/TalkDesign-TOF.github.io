博塔罗致 WNYE 的信

博塔罗给电视台 WNYE 写了一封信，关于他们安排播放“智能设计”宣传电影的事， 解开生命的奥秘.

by Andrea Bottaro

以下信件最初是发送给 WNYE PBS 电台的，以应对他们计划播出的智能设计“纪录片”《解开生命之谜》。最终，WNYE 播出了该节目，并附带了关于其内容的免责声明（参见 NCSE 网站 以获取 WNYE 免责声明的全文以及其他关于《解开生命之谜》的评论）。

Mr. Laing Kandel

总经理

WNYE

纽约

Dr. Joe Klein

校长

教育部

纽约


亲爱的坎德尔先生和克莱因博士：

最近，位于华盛顿州西雅图的发现研究所（Discovery Institute）宣布，纽约教育部的 WNYE 电视台计划于 2003 年 7 月 6 日播出纪录片视频《解开生命之谜》（以下简称 UML）。虽然我意识到 WNYE 的宗旨是为观众提供来自不同来源和观点的最广泛、最多样的信息，但我认为您和 WNYE 的观众应该了解关于该视频的一些背景问题。

UML 将自己呈现为一部制作精良、纯粹科学的纪录片，但在许多方面在事实上具有误导性，其主要目的是为一种被称为智能设计创造论（Intelligent Design Creationism）的伪科学运动进行宣传。UML 的起源（战略性地隐藏）接近宗教原教旨主义和创造论圈子，并显示出糟糕的学术风格，包括歪曲/遗漏关键的科学证据。最终，这些导致了事实、当前科学知识以及纪录片本身最终目标的误导性图景。（关于这些问题的更多细节见本信附件。）

虽然作为一名科学家，我完全支持来自任何来源的意见的自由传播，但我认为您有义务向 WNYE 的观众提供这些背景信息，以便他们能够正确判断纪录片的观点。在广播中添加免责声明，解释该纪录片呈现了一个边缘的、被绝大多数科学家拒绝的伪科学观点的一方面，并且其主要目的是宗教/哲学性质的，这可能会足以提醒您的观众《解开生命之谜》的真正意义。

非常感谢您的考虑。如果您有任何进一步的问题或需要更多信息，请随时与我联系。

此致

Andrea Bottaro

罗切斯特大学医学中心

纽约州罗切斯特，2003 年 6 月 30 日



解开生命之谜——更多是遗漏而非事实

来源：

"解开生命之谜"是名为"IllustraMedia"的实体制作的第一部也是唯一一部作品。事实上，"IllustraMedia"与"Discovery Media Productions"是同一个实体。1. Discovery Media 是一家制作公司，其之前的视频专注于福音派主题，如“天堂与地狱”和“末世”2. 虽然福音派视频公司制作科学纪录片本身并无不妥，但为了做到这一点，他们感到有必要创建一个“壳”制作公司，这凸显了整个企业笼罩在一种模糊的氛围中（更多例子随后）。此外，该视频作为宣传性和宗教性工具，而非科学/教育工具的目的，在其在基要主义圈子内的宣传方式中得到了强调。例如，《使命前沿》（Mission Frontiers），即福音派美国世界传教中心（U.S. Center for World Missions）的通讯，将其誉为“有史以来最令人印象深刻的传教工具”3.

内容如下：

作为一部纪录片，ULM 是一部技巧娴熟、制作精良的作品，展示了一些制作精良的分子水平细胞过程计算机动画。在讨论此类机制时，该视频声称科学证据表明标准进化论面临不可逾越的困难，并转而支持一个更高级的智慧直接干预创造和/或多样化生命的假设（因此得名“智能设计”，或 ID，创造论4). 该视频讨论了这些所谓的证据，并花费大量时间探讨 ID 运动的历史起源和哲学基础。

根本问题在于 ULM 是否符合基本的科学标准，即对证据和事实的遵守。在这方面，它在几个层面上失败了。首先，在整个纪录片中，由绝大多数科学家支持的主流科学观点甚至没有独立呈现。相反，ID 的支持者提供了对主流科学理论和假设的过度简化，有时甚至是公然轻蔑的阐述（作为一个反例，最近的 PBS“进化”系列虽然明显倾向于科学观点，但也收录了几位著名创造论代表的意见）。因此，在 ULM 中，观众看到的是与科学界实际情况几乎没有联系的科学证据和理论描述。出于篇幅原因，我只列举几个例子。

最明显的遗漏涉及 ULM 对生命起源（OoL）科学的讨论。视频中讨论的唯一非 ID 基础的 OoL 观点是由当前 ID 支持者之一、肯恩博士（Dr. Dean Kenyon）在 20 世纪 60 年代后期提出的。根据 ULM，肯恩及其同事后来证明这些模型不足以解释早期分子和细胞进化的关键方面。事实上，肯恩的许多原始观点早已被更彻底、且更有实证支持的科学假设所取代——正是这些假设和证据导致了肯恩的思想在科学界在 ID 创造论出现之前很久就走向终结。可悲的是，目前（并且已经十多年了）关于 OoL 的 favored 假设，即所谓的“RNA 世界”模型，5 在 ULM 中完全找不到提及。这并不奇怪，也许是因为 ID 支持者在 ULM 中对肯恩原始理论提出的反对意见，并不能针对这个新模型成立。因此，观众被给予了错误的印象，认为当前的科学选择是在 ID 创造论及其彻底的奇迹生命起源，或者肯恩博士过时的 20 世纪 60 年代理论之间。当然，我们对 OoL 的理解仍然非常有限，且高度推测性。然而，它比 ULM 的拙劣模仿希望其受众相信的要先进和科学严谨得多。

UML 中的其他错误还包括对“借用”（cooption）这一至关重要的进化机制进行了同样肤浅、近乎戏谑的简化讨论，而在生物界实际上存在支持其重要性的证据。UML 的“专家”甚至犯了一个关于核酸在细胞中作用的基本错误，即将其描述为唯一参与遗传信息存储和转移的分子，而众所周知，它们直接参与关键的分子过程，其功能与蛋白质酶所执行的过程相当——这是支持上述“RNA 世界”假说的关键证据（也是其未被提及的可能原因）。

整个智能设计（ID）哲学背后的核心论点，在视频中得到了广泛讨论，即“不可还原复杂性”（irreducible complexity）系统的概念，以及所谓传统进化机制无法产生这些系统的不可能性。尽管该论点很快就被生物学家在理论和实证基础上予以驳斥，6 "不可还原复杂性"（irreducible complexity）一直是 ID 创造论的主要支柱。讽刺的是，这一论点最近在一本 prestigious 的科学期刊《自然》（Nature）中受到了致命打击，该期刊发表了一项完全基于进化原则（无定向随机突变和选择）的计算机模拟，证明其能够生成"不可还原复杂性"（irreducibly complex）的输出结果7。虽然当然不能责怪视频未能预测未来科学研究的结果，但这一事件很好地说明了 ID 推理所依据的根基是多么脆弱。事实上，不仅支持 ID 的科学证据继续积累并与其论点相矛盾，而且更令人痛心的是，在“运动”开始后的 10 多年里， 没有任何支持 ID 的科学结果发表在科学文献中，尽管其支持者继续声称存在此类结果。事实上，就连 ID 倡导者自己的期刊《复杂性与信息设计进展》（Progress in Complexity, Information and Design）的电子出版物，也未能发表任何支持 ID 的实验结果。8.

简而言之，尽管 ID 倡导者呼吁“让证据自己说话”，但实际上并没有支持 ID 的正面科学证据，相反，其倡导者的理论论点在专业文献中不断被证明是错误的。为了避免面对这一缺乏证据的事实，UML 转而采取了对主流科学理论的系统性歪曲以及对关键思想和证据的遗漏。

人物：

接受 UML 采访的专家以及 ID 倡导者通常喜欢将自己描述为“科学家”，并经常加上“虽然人数不多但在增长”的限定词。事实上，大多数 ID 倡导者在任何有意义的定义下都不是科学家，而且他们的数量（“不多”这一说法本身就是夸大）远非在增长。

在 UML 中出现的专家中，有 4 位确实可以被视为真正的科学家：迈克尔·比赫（Michael Behe）、斯科特·米尼奇（Scott Minnich）、迪恩·凯尼恩（Dean Kenyon）和杰德·马科斯科（Jed Macosko）。前两位分别在主流大学的生物化学和微生物学领域拥有终身教职，但正如上文讨论的那样，尽管他们拥有自己的研究经验和活跃实验室，却未能提供任何证据来支持他们如此雄辩地主张的观点。迪恩·凯尼恩在 20 世纪 70 年代中期之前一直从事科学活动，此后未在科学文献中发表过任何文章（然而，他后来合著了臭名昭著的创造论学校教科书《熊猫与人类》）。9). 16 杰德·马科斯科在 UML 中的图片旁附有“分子生物学家，加州大学伯克利分校”的限定词，尽管他是伯克利毕业生和前博士后研究员，但他实际上从未（或现在）是伯克利教职员工，正如该头衔可能暗示的那样。事实上，马科斯科博士似乎甚至不再与加州大学伯克利分校有关联；如果他在采访当时确实在那里，他肯定是一名初级博士后研究员，按照任何标准都算不上该领域的“专家”。目前，马科斯科博士在一些 ID 网站上被列为在加利福尼亚州河滨市的宗教拉西尔亚大学（La Sierra University）教授化学。10尽管他也没有出现在那里的教职员工名单上。11这种“慷慨”地使用资历的情况在纪录片中并非独一无二。ID 的主要倡导者之一威廉·德姆斯基（William Dembski）在 UML 中被标记为“数学家 - 贝勒大学”，尽管他与贝勒大学的信仰与学习研究所（Institute for Faith and Learning）有关联，该研究所专注于神学和哲学。12事实上，德姆斯基博士庞大的出版著作中，除了 1990 年的一篇数学论文外，其余几乎全部是关于神学、护教学和哲学的各个方面。13 （德姆斯基博士拥有数学和哲学博士学位，以及神学神学硕士文凭）。最后，乔纳森·韦尔斯（Jonathan Wells）在 UML 中被介绍为“生物学家”，他确实拥有加州大学伯克利分校发育生物学博士学位。但据他本人所述，他进入该项目并非基于对科学和生物学的真正兴趣，而是遵循了鲜文教会长（Rev. Sun Myung Moon）的指示，其明确目标是“将毕生精力投入到摧毁达尔文主义上”。14毫不奇怪，在韦尔斯博士接受伯克利培训后，没有任何记录显示他进行了任何有意义的研究工作，此后他完全致力于反进化论宣传（包括书籍《进化偶像》，其中一些版本甚至包含让学生涂污生物学教科书的贴纸）。15).

因此，UML 全文中使用的专业背景与学术隶属关系的定义至少是模糊的，并显然导致参与者的表面学术影响力及相关专业知识的虚高。

总之，《解开生命之谜》描绘的是一种至多半科学性质的边缘哲学运动，该运动在意识形态和组织结构上与宗教创造论极为接近。该纪录片在多个重要方面歪曲了其自身、其目标、现有的科学证据以及其自身的专家。虽然播放您认为最适合观众需求和兴趣的节目是您的权利，但同样重要的是，您的观众应获得有助于将其将该产品内容及其目的置于适当背景中的信息。这不仅出于开放和充分披露的精神，也是为了避免您的纪录片播出被视为该国最大教育部门之一对该新型“隐形”创造论的隐含认可。
