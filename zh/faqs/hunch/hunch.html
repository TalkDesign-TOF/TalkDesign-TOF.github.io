"The Quixotic Message" 或 "No Free Hunch"

Steve Reuland（又名 "theyeti"）以幽默但引用详实的方式审视了智能设计运动及其倡导者之间的矛盾。

by theyeti *

[贡献于 2003 年 1 月 22 日]

[作者注： 这个矛盾和荒谬的集合最初是在 antievolution.org 讨论板上编写和发布的。后来决定将其发布到网上，以便让更广泛的受众看到。这个集合旨在轻松幽默，尽管它确实突出了 ID 辩论中的一些严肃问题。如果这让你感到不适，那就去读点别的东西。此外，应几位读者的敦促，我决定制作一个参考文献页面来记录这些不一致之处。可以通过点击下面任何一个编号的参考文献来访问。如果你觉得这会破坏幽默感，那就随意跳过它。但正如 Dave Barry 所说， I'm not making this up...]

IDists...

On Intelligent Design...

  • ID 是我们说什么就是什么，我们并不一致。

  • 越来越多的科学家加入了 ID 运动，这就是为什么我们每年都要重复引用同样的三个来源。[]1]

  • ID 并非创造论，且可以与进化论完美兼容。这就是为什么我们要要求学校教授“反对进化论的证据”。2]

  • 我们不是创造论者，除了那些我们自己是的，但我们其余的人不会确认我们不是。但如果你称我们为创造论者，我们会不停地抱怨。3]

  • 关于古老地球、生物共同祖先和自然选择等问题的正确立场，可以在我们说服公众他们应该至少拒绝其中一项之后再确定。4]

  • ID 是设计科学领域内被广泛接受的理论。就在去年，超过 100 位科学家签署了一份声明，该声明不支持 ID，但表示他们对达尔文主义持“怀疑”态度。其他成千上万科学家的意见不算数，因为他们都有偏见。5]

  • ID 是一个用于研究设计科学的计划，仅此而已。我们研究计划的一部分是为学龄前儿童制作涂色书，并在聚会上让自己更讨人喜欢。6]

  • ID 是一个用于检测自然界中目的和目的论的科学理论。但不要问我们那个目的是什么，因为那是一个与 ID 无关的宗教问题。

  • 设计者可能是从上帝到太空人。但相信设计者是太空人的拉利派（Raelians）是不合逻辑的。

关于达尔文主义...

  • 达尔文主义无法解释生命演化的每一个细节，因此它是错误的。但请不要要求智能设计论者（IDists）解释这些事情，因为那不是智能设计（ID）这种理论应该做的。[7]

  • 主流科学家不敢与达尔文正统学说这一铁板一块相左。然而，这里有一些主流科学家在某些问题上彼此意见不一，这意味着他们无法达成一致。[8]

  • 达尔文主义者受宗教和意识形态动机的驱使。但既然我们已经从我们的网站上移除了上帝的图片和“文化复兴”的短语，大家就知道这不适用于我们。[9]

  • 社会中的一切错误都是由教条的达尔文主义无神论唯物主义者造成的。包括刻板印象、妖魔化和替罪羊。[10]

  • 达尔文主义者既对社会主义和自由放任资本主义负责，也对种族主义和自由主义负责。既对女权主义和性别歧视负责，也对动物实验和动物权利运动负责。还有共产主义纳粹主义。[11]

关于哲学...

  • 哲学家无法就科学与非科学之间的确切界限达成一致。因此，如果我们说某事是科学，那么任何事都可以被认为是科学。

  • 如果一个生命系统看起来设计良好，那就是智能设计（ID）的证据。如果它看起来设计得很差，那只是因为我们无法知道什么是好设计，什么是坏设计。

  • 毕竟，我们无法判断这是否是糟糕的设计，因为我们无法知道设计者真正的意图。但我们确实知道，根据设计者的意图，智能设计（ID）将彻底改变文化、社会和法律。[14]

  • 方法论自然主义是一条不公平的规则，阻止我们考虑超自然解释。但这意味着侦探在调查人员死亡时不能考虑智能代理，因为众所周知，凶手是超自然的。[15]

  • 一个好的科学理论，如智能设计（ID），应该是模糊和模棱两可的，并且拒绝提出任何关于机制或历史的特定细节。某个未指明的存在在某个时间点、某个地方、以某种方式“设计”了某物，这是一个完全合理的解释。

  • 设计论的论证不是一个神学论证，因为我们不一定在谈论上帝。但任何对设计论的反驳都是神学的，因为它要求我们说“上帝不会这样做”，这是不合法的。[16]

关于证据...

  • 既然胡椒蛾的案例已被证明存在问题，自然选择就被推翻了。野外对自然选择的另外 1,582 项研究，以及大量的实验室研究，都不算数。17]

  • 而且胡椒蛾不会停在树干上。在野外自然位置发现的蛾子的实际数据集，无论是在树干上还是不在树干上，都是无关紧要的，因为研究人员多年来在捕蛾器中捕获了数千只蛾子，从未发现一只捕蛾器中的蛾子停在树干上。18]

  • 既然蛾子不会停在树干上，而是停在更高的树枝上，这意味着鸟类无法到达它们，因为有一个魔法屏障阻止鸟类访问树枝。

  • 如上所示，蛾子并不停在树干上，这意味着教科书中展示蛾子在树干上对比鲜明度的照片是欺诈，欺诈，欺诈。然而，教科书中所有其他摆拍的动物照片都是无可指责的。

  • 更包容的分组（如门）出现在更具体的分组（如属）之前，这是反对进化的证据。同样，欧洲人出现在托尼·布莱尔之前这一事实也是反对共同人类祖先的证据。 [19]

  • 进化无法产生新信息，因为任何增加酶底物特异性的变化都会降低酶与其他化合物的反应性，这是一种信息损失。同样，任何增加酶通用性的变化都是信息损失，因为酶失去了一些特异性。 [20]

  • 生命不可能通过自然方式产生，因为它具有指定复杂性。指定复杂性意味着无法通过自然方式产生的事物，因此生命必须表现出指定复杂性。 [21]

  • 我们要感谢我们慈爱的设计者设计了免疫系统来保护我们免受他设计的致命疾病的侵害。

  • 生命的根本统一性意味着只有一个设计者。生命之间（包括它们相互杀戮的倾向）的非凡差异，仅仅意味着我们唯一的设计者非常富有创造力和 whimsical（古怪）。 [22]

  • 侧向基因转移是进化的强大机制，却是反对进化的证据。

  • 宇宙定律完美适合生命这一事实是存在设计者的证据。宇宙定律无法产生生命这一事实也是存在设计者的证据。 [23]

  • 不可还原复杂的结构需要多个部分。因此它们无法进化。如果有人证明需要多个部分的结构是如何进化的，那只是意味着它并不是不可还原复杂的。

  • IC 结构必须是分子系统。除了捕鼠器。

  • "间接"路径极不可能，且难以找到，就像找到小精灵一样，因此它们仅仅是"勉强"的可能性，而非现实的可能性，尽管每一位从达尔文到道金斯的生物学家都对其给予了详细关注，但仍可安全地予以忽视。[26]

  • ID 假设另一方面，与小精灵毫无相似之处。
